# H5P.Anki (IDEA)

You choose a content type among a list of supported content type and it diplay a button, if you click on it it add cards to anki based on the content type

## Supported content type

- Dialog Card ⭐⭐⭐
- Flash cards ⭐⭐⭐
- Image pairing ⭐
- Memory game ⭐
- Multiple choice ⭐⭐
- Single choice set ⭐⭐
- True / false ⭐
